* Play Information *

================================================================
Copy the ztn2dm5.bsp to your Quake2\baseq2\maps directory, run
Quake II, pull down the console and enter: map ztn2dm5
================================================================
There's also a route table for the Eraser bot bundled with this
map. Copy the ztn2dm5.rtz file to your Eraser\routes directory 
and your Erasers will be ready to go. Eraser bot can be obtained
from http://impact.frag.com
================================================================

Single Player           : No
Cooperative             : No
Deathmatch              : Yes, 7 respawn spots
Player load			: Yes, 2 - 6
Difficulty Settings     : No
New Sounds              : No
New Graphics            : No
