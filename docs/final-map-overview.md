# Final Map Overview

This overview covers the final remastered maps in `finals/maps`. Long map names come
from each BSP's worldspawn `message` key, and original metadata is based on the Discord
release posts. Levelshots reference the first image in `media/` when available. All
maps are intended for the latest version of [Muff Mode](https://github.com/themuffinator/muffmode).
