* Play Information *

 Settings                : enable "weapons stay" for 5 or more players
 Deathmatch 2-8          : at any rate! 2-on-2 recommended
 Deathmatch 9-100        : maybe not...


* Construction *

 Build Time      	 : 4 weeks.
 Compile Time    	 : less than 3 minutes, even without detail brushes.
 Editor(s) used  	 : Worldcraft
 Known Bugs      	 : none.


* Other Info *
 
 There's a MH hidden behind a door in the atrium with the button.
 Just push the button or shoot the door (once you know where it is.)
